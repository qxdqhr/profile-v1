# 小红书官方开放平台

## 入口

- 开放平台文档：https://open.xiaohongshu.com/
- API 调试（Apifox 镜像）：https://xiaohongshu.apifox.cn/

## 定位

官方 API **面向电商与服务商**，非个人创作者内容发布。典型接入方：ERP、打单工具、商品搬家、订单管理系统。

## 类目与权限包（节选）

| 类目 | API 权限 | 是否含发帖 |
|------|----------|------------|
| 一键搬家 | 公共、授权、商品、库存、素材 | ❌ |
| 打单工具 | + 订单、售后 | ❌ |
| 订单管理 | + 素材中心 | ❌ |
| 企业 ERP | 全商品/订单包 | ❌ |
| 商家后台系统 | 所有权限包 | ❌ |

权限包关键词：**商品、库存、订单、售后、素材中心**——均无「笔记发布」「创作者内容」类接口。

## 接入条件

1. 注册开发者账号
2. 创建应用（每类型限一个）
3. 获取 `appId` / `appSecret`
4. OAuth 授权店铺
5. 按类目申请对应 API 包

## 结论

个人或副业场景下的「自动发帖」**无法**通过官方开放平台实现，需走 Web Cookie 或浏览器自动化路线（见 `02-web-cookie-flow.md`）。
